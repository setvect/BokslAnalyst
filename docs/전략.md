# 전략 

## 1. 가치평가 전략(한국 주식)

- 메인 소스:
- 실행 소스:
- 매수조건
    - 필터
        1. 기타금융, 생명보험, 손해보험, 은행, 증권, 창업투자, 우선주 제외
        2. PER 양수, PBR 값이 있는 종목
        3. 시총 순위 70% ~ 90% 사이 기업
    - 순위 매김
        - 1/PER
        - 1/PBR
        - 배당수익률
    - 각각의 등수를 더해 오름차순 정렬
    - 후처리(수동)
        - 중국기업 제외
        - 지주회사 제외
        - 기업명에 `홀딩스` 포함 기업제외
        - 전년도 영업이익 적자 제외 OR 직전 2분기 연속 영업이익 적자 제외
    - 상위 20개 기업 매수
- 매도 조건
    - 1년에 한 번 또는 두 번 리벨런싱
